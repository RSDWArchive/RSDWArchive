Run [`generate_npc_wikitable.py`](generate_npc_wikitable.py) to build a sortable wikitable from [`../../tools/NPCData/NPCData.json`](../../tools/NPCData/NPCData.json) (default output: [`npc_wikitable.txt`](npc_wikitable.txt)). Each row uses the same **Image** / **Name** layout as [`../icons/generate_icon_wikitable.py`](../icons/generate_icon_wikitable.py) (`[[File:{title}.png|…]]` and `[[{title}]]`); PNGs are not verified on disk. Display names are de-duplicated and title-cased the same way as icons. This is done on purpose. Valdes takes pictures of each NPC manually because he's crazy.
