# MapData Experiments

This folder is used for experimenting with interactive map support for `LocationData.html`.

Current prototype:
- Uses Leaflet (`cdnjs.cloudflare.com/ajax/libs/leaflet/1.9.4`)
- Uses Dragonwilds tiles:
  - `https://maps.runescape.wiki/dw/tiles/{z}/{x}_{y}.png`
- Uses the same CRS transformation approach as the wiki gadget.

Related wiki references:
- `https://dragonwilds.runescape.wiki/w/Module:Map`
- `https://dragonwilds.runescape.wiki/w/Template:Map`
- `https://dragonwilds.runescape.wiki/w/MediaWiki:Gadget-maps.js`
